# ✅ Habit & Mood Tracker

A lightweight, full-stack productivity application that helps users build consistent habits while tracking their daily mood. The platform enables users to log activities such as workouts, studying, hydration, and meditation, alongside a daily mood rating. By visualizing historical trends through interactive charts, users can identify how specific habits influence their overall well-being and maintain healthier routines.

---

## 🧱 Technology Stack

| Layer           | Technology                                         |
| --------------- | -------------------------------------------------- |
| **Frontend**    | Streamlit (Python-based responsive user interface) |
| **Backend/API** | Flask, Flask-SQLAlchemy                            |
| **Database**    | PostgreSQL                                         |
